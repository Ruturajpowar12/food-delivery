# Food Ordering Web App (MERN Stack)

A simple **Food Ordering Web Application** built using the **MERN Stack**. Users can browse food items, add them to the cart, and place orders.

---

## Features

* User login & registration
* Browse food items
* Add to cart & place orders
* Order management
* Responsive design

---

## Tech Stack

**Frontend:** React, Vite, Tailwind CSS
**Backend:** Node.js, Express.js
**Database:** MongoDB
**Authentication:** JWT
